About Me

I am an incoming Ph.D. student in Computer Science and Technology at the Institute of Computing Technology, Chinese Academy of Sciences, advised by Prof. Jian Wang. I will join the Microprocessor Research Center and the State Key Lab of Processors, ICT, CAS in late August 2026.

My research interests are computer architecture, chiplet interconnects, memory hierarchy, and unified shared memory for heterogeneous systems. I received my B.Eng. degree in Electronic Information Engineering from ShanghaiTech University.

Computer Architecture Chiplet Interconnects Memory Hierarchy Unified Shared Memory

Research Interests

  • Computer Architecture: processor architecture, RISC-V systems, memory hierarchy, cache behavior, cache coherence, and performance modeling.
  • Chiplet Interconnects: die-to-die interfaces, NoC/NoP fabrics, UCIe-like links, package-aware bandwidth/latency/power tradeoffs, and heterogeneous chiplet systems.
  • USM-Aware Memory Fabric: shared and coherent memory for heterogeneous chiplets, CXL-like memory semantics, device-initiated access, region-based coherence, and page/object placement.
  • Architecture-IC Co-Design: using digital IC and analog/mixed-signal circuit background to reason about timing, power, area, link reliability, and practical implementation constraints.
