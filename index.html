About Me

I am an incoming Ph.D. student at the Institute of Computing Technology, Chinese Academy of Sciences, advised by Prof. Jian Wang. My interests lie in computer architecture, chiplet interconnects, and memory systems for heterogeneous computing.

Computer Architecture Chiplet Interconnects CXL / UCIe Memory Hierarchy Unified Shared Memory

Research

My current focus is link-aware memory fabrics for heterogeneous chiplet systems. I am interested in how realistic D2D, CXL, and UCIe-like link behavior affects shared memory, coherent access, and unified shared memory design.

  • Chiplet interconnects: NoC/NoP fabrics, D2D links, and CXL/UCIe-like systems.
  • Memory systems: cache coherence, shared memory, USM, and device-initiated access.
  • Modeling: lightweight simulation and performance analysis for architecture policies.
